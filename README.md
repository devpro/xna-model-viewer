# Xna Model Viewer

This is a simple project to use XNA framework in order to load and view 3D models.

## Installation

[MonoGame/MonoGame](https://github.com/MonoGame/MonoGame)

```bash
dotnet new install "MonoGame.Templates.CSharp"
```
